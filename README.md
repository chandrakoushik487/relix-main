# Relix
Intelligent Volunteer and Resource Coordination Platform

## Overview
Relix is a data-driven system that efficiently matches volunteers and allocates resources for social impact initiatives and emergency response.

## Key Features
- Smart volunteer to task matching  
- Location-based allocation  
- Emergency response mode  
- Resource management  
- Admin dashboard  

## How It Works
1. Volunteers register with their details  
2. Admin creates tasks  
3. System assigns the best suitable volunteers automatically
