— How it works

A four-step pipeline from paper to priority action.

Upload

NGO workers photograph handwritten forms and submit them through the RELIX intake flow.

AI Extraction

OCR + LLM extracts location, urgency, and issue type. Google Vision API is primary and Tesseract serves as fallback.

Prioritize

SVI scores classify reports as Critical, High, Medium, or Low using urgency (60%) and scale (40%).

Coordinate

The dashboard assigns volunteers, tracks resolution, and keeps teams aligned in real time.

— Features

Operational capabilities for every disaster response team.

Real-Time Dashboard

Social Vulnerability Index (SVI)

Weighted scoring uses urgency (60%) and scale (40%). Location clustering, density multipliers, and auto-recalculation highlight the most vulnerable areas.

Auto-score 4-tier system Real-time

Proximity-Aware Volunteer Dispatch

Haversine distance scoring, skill matching, and workload balancing support auto-assignment for critical needs and admin confirmation for lower priority cases.

— About / Team

RELIX was built to ensure no community goes unserved during a disaster because their voices couldn't be digitized fast enough.

RELIX brings editorial clarity to emergency response workflows. Designed for NGO workers, field coordinators, and volunteers, it turns ambiguous paper reports into structured action so that every report can be routed, scored, and resolved without delay.
